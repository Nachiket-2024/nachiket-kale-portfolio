# Nachiket Kale Portfolio

Personal portfolio website showcasing my software engineering experience, projects, education, and resume.

Built with Next.js, TypeScript, Tailwind CSS, and MDX.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- React
- Tailwind CSS
- MDX

## Features

- Responsive portfolio website
- MDX-based content management
- Dynamic project pages
- Resume PDF download
- SEO metadata configuration
- Sitemap and robots.txt generation
- Static generation with Next.js

## Deployment

Deployed using Vercel: https://nachiket-kale-portfolio.vercel.app
