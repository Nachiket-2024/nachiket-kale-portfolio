## Overview

Balance Sheet Analysis is a web application for importing, storing, and exploring company financial data.

The application retrieves balance sheet data from Yahoo Finance, processes and stores financial records in PostgreSQL, and exposes the data through FastAPI APIs with a React frontend.

The project explores integrating external financial data sources with authentication, database workflows, and user-facing applications.

## Highlights

- Integrated Yahoo Finance data retrieval and built workflows for importing and managing company balance sheet records.
- Developed FastAPI APIs with PostgreSQL storage for structured financial data access.
- Implemented Google OAuth2 authentication, JWT-based authorization, and role-based access control for protected operations.
- Added an experimental natural language query feature using an LLM API to explore financial data interactions.
