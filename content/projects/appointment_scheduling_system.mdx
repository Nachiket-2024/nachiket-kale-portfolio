## Overview

Appointment Scheduling System is a web application for managing doctor appointments with separate workflows for administrators, doctors, and patients.

The system supports OAuth2/JWT authentication, role-based access control, appointment scheduling, doctor availability management, and automated notifications through Google Calendar and Gmail APIs.

The backend follows a modular structure with separate authentication, scheduling, and notification services to keep application logic organized and maintainable.

## Highlights

- Built role-based workflows for Admin, Doctor, and Patient users with permissions enforced across appointment and user management operations.
- Implemented OAuth2/JWT authentication with protected API routes and user-specific access control.
- Integrated Google Calendar and Gmail APIs for appointment synchronization and automated email notifications.
- Developed appointment scheduling workflows including doctor availability, booking, rescheduling, and cancellation handling.
