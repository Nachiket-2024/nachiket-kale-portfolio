## Overview

MysticAuth is an authentication and authorization system built to explore secure user management and access control workflows.

The system supports email/password authentication, Google OAuth2 with PKCE, JWT-based session management, refresh token rotation, and fine-grained authorization through Policy-Based Access Control (PBAC).

The project follows a feature-based structure with clear separation between API routes, business logic, repositories, and authorization components.

## Highlights

- Implemented authentication workflows including email verification, password reset, refresh token rotation, and session revocation.
- Built Policy-Based Access Control (PBAC) to separate authorization decisions from user roles, enabling flexible permission management.
- Implemented refresh token rotation with reuse detection to improve session security.
- Structured the application using feature-based modules with separation between API routes, business logic, repositories, and authorization components.
- Developed backend and frontend test suites with 650+ automated tests, CI validation, and detailed technical documentation.
