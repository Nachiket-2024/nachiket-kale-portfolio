## Overview

Secure UPI Gateway is a backend simulation of a digital payment system exploring transaction processing and cryptographic security concepts.

The system provides APIs for user and merchant management, payment workflows, QR-based transactions, and maintaining transaction records through a ledger-based approach.

## Highlights

- Built FastAPI backend services for user accounts, merchant management, and transaction processing workflows.
- Implemented cryptographic mechanisms including SHA-256 hashing, PBKDF2 password protection, and AES-based data encryption.
- Developed QR-based payment flows with transaction verification and ledger-style record tracking.
- Explored lightweight cryptography concepts through a simulated secure payment environment.
